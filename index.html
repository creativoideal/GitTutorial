Just Remember This Commands

  • git clone "URL" will clone the reporitory to your computer
  • git add -A will prepare your changes
  • git status will check the local satus
  • git commit -m "MESSAGE" will add a message to the changes you made
  • git push will push your changes to github
  • git pull will pull new changes from gitbut
